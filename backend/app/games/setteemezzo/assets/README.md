# Assets - Sette e Mezzo

## Spritesheet del Croupier

Posiziona i seguenti file in questa cartella:

- `croupier_idle.png` — Spritesheet orizzontale dell'animazione IDLE del croupier
- `croupier_deal.png` — Spritesheet orizzontale dell'animazione di distribuzione carte

### Formato spritesheet
- Striscia orizzontale (tutti i frame affiancati)
- Il sistema auto-rileva il numero di frame assumendo frame quadrati
- Se i frame non sono quadrati, specificare `frameCount` nel costruttore
